Important Process
  • Download data Template.
  • Save file in .xls Format.
  • Input the data in excel and then upload.
  • Don't change the header contents.
  • Please input Date of Training in 'YYYYMMDD' format.(ex. 20190611)
  • GTM Code column can not be left blank in case designation is Channel Partner.
  • GTM Code should be a 7 character length containing first 3 character as Alphabets only and last 4 as numeric only.(ex. ABC1111)
  • Training Medium column should not be left blank and contain value either Classroom or On-Store or Virtual only. Other kind of value is not allowed.
